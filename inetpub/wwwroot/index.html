To customise the install edit the Response.json file under the Visual Studio version and take out all:true and use the add section instead, there is an example to add already in the Response.json file but it is overridden at the moment by the all:True setting

To make sure you can run the powershells scripts for the Microsoft Visual Studio 2022 Offline Installer, run up the powershell IDE in Admin mode and use the commands

Set-ExecutionPolicy remotesigned -scope CurrentUser
Then press enter (pwsh prompt) or F5 in PowerShell IDE

Set-ExecutionPolicy remotesigned -scope LocalMachine
Then press enter (pwsh prompt) or F5 in PowerShell IDE
